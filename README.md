Secure Expense Manager

A Java-based personal expense and income management system with secure authentication, MySQL database, and budget tracking.

Track your expenses, add income, view monthly totals, get alerts for exceeding your budget, and maintain a secure personal finance record.


Features:
1.User Authentication
  Secure registration and login using hashed passwords.

2.Expense Management
  Add, view, and delete expenses. Track category, amount, date, and notes.

3.Income Management
  Add income records with source and amount.

4.Monthly Summary
  View monthly totals of income and expenses.

5.Budget Alerts
  Automatic warnings when expenses exceed income for the month.

6.Secure Database Access
  Uses prepared statements to prevent SQL injection.


Tech Stack:

-->Language: Java (JDK 17+)

-->Database: MySQL

-->Database Connectivity: JDBC

-->Password Security: SHA-256 hashing

-->IDE/Editor: VS Code (recommended)
